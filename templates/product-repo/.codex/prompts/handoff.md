# /handoff

Provide a concise handoff:

- Scope/status: done, pending, blocked.
- Git: branch, `git status --short --branch`, local commits, pushed state.
- Issue/PR: URL, milestone, base branch, CI state.
- Validation: commands run, live proof, unrun checks.
- Files changed and rationale.
- Next steps in order.
- Risks/gotchas, credentials, local URLs, feature flags, or release notes.
