# /pickup

Use `$wp-product-orchestrator`.

1. Read `AGENTS.md`, `PRODUCT.md`, release docs, and relevant issue/PR if present.
2. Check `git status --short --branch`, remotes, current branch, release branches, local commits, and open PR for the branch.
3. Identify the current mode: triage, implement, review, release, or handoff.
4. Load only the needed shared reference and one `wp-expert` lane.
5. Report next 2-3 actions and execute if already authorized.
